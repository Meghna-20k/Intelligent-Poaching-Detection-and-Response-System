# Intelligent Wildlife Poaching Detection System

## Overview
This project aims to detect suspicious human activity in wildlife areas using deep learning and computer vision techniques.

## Problem Statement
Wildlife poaching is a major threat. Manual monitoring is inefficient. This system automates detection using AI.

## Approach
- Used deep learning model for image-based detection
- Processed input images/videos
- Classified presence of human activity

## Technologies Used
- Python
- PyTorch
- OpenCV
- Google Colab

## Results
- Successfully detected human presence in test inputs
- Generated predictions with confidence scores
